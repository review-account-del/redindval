# redindval
Electronic Supplemental Materials (ESM) for the paper "RedIndVal: a composite indicator for identifying species associated with high conservation value communities" as a review repository for Python scripts

To run the pipeline: start to download/clone the deposit and arrange a folder structure. 
        
Open the Python scripts in numeric order. Update the folder- and file paths in CONFIG, but be sure not to change the name of specified files. Run each script in numeric order. Note: the database of occurrence data (stored in occurrence_ds) is on GitHub just a sample due to large files. Thus, the output from the pipeline will not be exactly as the results in the main paper.
